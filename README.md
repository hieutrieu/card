# Flower Color Test

## Description

- Choose of 3 colors out of 8 colors.
- Each color is represented as a flower-textured card.
- Smartphone & PC compatible.
- Fanpage: https://www.facebook.com/MegumiHarunaJewelry | ID 1441148306133126

## Animation of Card

- Before choice: Cards are flower-textured
- After choice: Turned into gems + planets

## The pages

1. Introduction Page
2. Choose Page
    3 cards flippable.
    After the choice of 3, next button is enabled on the bottom.
3. Like Page
    Show the like button. After user liked, show the result page.
4. Result Page
    Show Recommended Jewelry Pic on E-commerce site with the chosen card.
    Show Share On Facebook.
    
## How to deploy the code

1. Create facebook applications. Get the ID and place to the code.
2. Submit for login review. To use user-likes permission.
